TLVR

Rhodes Island Tetralingual Voice Record Archive

Welcome home, Doctor!

TLVR is an Arknights fansite that makes it easy to browse voice lines from the game in all its four languages (English, Chinese, Japanese, Korean). It’s created with multilingual speakers and language-learners in mind!

This site includes all characters and all voice files currently available in the Global servers, including regional voices. Check footer for when the site was last updated.

This site was made with love by PseudoMon. Its creation was inspired by ASTR. The site's source code is available on GitHub.

If you found a bug or have suggestions for the site, you can submit an issue in Github or contact me directly on Twitter or Mastodon.
